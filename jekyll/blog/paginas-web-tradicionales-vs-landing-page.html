Páginas web tradicionales vs Landing Pages

10/09/2016

A fines de los años 90 las páginas web se diseñaban desde el punto de vista artístico. La prioridad era que una página web sea agradable a la vista asi como lo es un cuadro en una exposición de arte. Como veremos más adelante a medida que va evolucionando Internet esta mentalidad fue cambiando.

Aproximadamente en el año 2004 surge el término Web 2.0 que significa que no es suficiente que una página web sea agradable a la vista sino que lo más importante de una página web es que sea fácil de utilizar por el usuario y cumpla con el objetivo para el cual fue diseñada.

Antes de empezar el proyecto de página web para tu empresa te tienes que preguntar lo siguiente. ¿Qué es lo que buscamos cuando un visitante entra a nuestra página web? El 99% de las veces la respuesta es clara. Tu quieres aumentar las ventas de tu empresa y hacer que tu negocio crezca. Ahora viene lo interesante.

Una Landing Page es una página web diseñada para cumplir con un objetivo bien preciso. Es decir se diseñada la Landing Page poniendo como prioridad la usabilidad de los elementos presentes en la página web para que cada uno de estos ayuden cumplir el objetivo.

Básicamente existen dos tipos de objetivos en una Landing Page: Click Through y Generar contactos. Si quieres saber más puedes leer el artículo ¿Qué es una pagina vendedora?

Ventajas de una Landing page contra una Página web tradicional

Conversiones

Una Landing Page convierte un visitante en un prospecto (Potencial cliente)

Cuando un visitante entra a una Landing Page queremos que ejecute una acción bien definida. Para el 99% de las pequeñas empresas esta acción es que el visitante nos contacte ya sea por teléfono o por un formulario. Una vez que teniendo el contacto podemos transformar el visitante en un cliente. En ese momento estás recuperando tu inversión en la Landing Page.

En una página web tradicional el visitante tiene muchas opciones que lo distrae de tu objetivo. Por ejemplo un error común de una página web de una pequeña empresa es el de publicar su misión, visión y historia. La verdad es que a nadie le interesa, lo que los visitante quieres es saber es que vendes, qué servicios das y como contactarte. En una página web tradicional el 99% de los visitante simplemente abandona la página sin tomar ninguna acción o sea para decirlo fríamente perdiste un potencial cliente.

  • Landing pages convierte visitante en clientes
  • Páginas webs tradicionales hacen perder el tiempo (dinero)

SEO

Las reglas básicas de SEO nos dice que tienes que segmentar tus palabras claves lo más posible. El contenido genérico es como el veneno para Google. Simplemente Google no va entender de que habla tu página web y no la va a mostrar en los primeros resultados.

Como explicamos antes una caracteristica de una Landing page es que debe cumplir con un objetivo bien preciso que tiene como consecuencia que podemos optimizar las palabras claves para obtener un buen resultado del SEO Orgánico y de pago. En palabras simples una Landing Page tiene más tráfico y de mejor calidad.

Una página web tradicional es por su diseño muy genérica, se habla de la historia de la empresa, de la misión, del dueño, etc… es decir contiene muchas palabras claves que se canibalizan entre sí y trae como consecuencia que Google penaliza tu página simplemente porque entiende que es lo que ofrece. Una página web tradicional va tener poco tráfico y de poca calidad.

  • Landing page obtienen más tráfico y de mejor calidad
  • Una página web tradicional poco tráfico y muy genérico dificil que convertir

Para una pequeña y mediana empresa la mejor solución para hacer crecer el negocio por Internet es una Landing Page.
